# Web Dashboard

This folder contains a lightweight static dashboard version of the project.

## Files
- `index.html`: Main dashboard page
- `styles.css`: Visual styling

## How To View
- Open `index.html` directly in a browser
- or enable GitHub Pages for the repository and use this folder as the published source

## What The Dashboard Shows
- headline metrics
- funnel narrative
- desktop versus mobile comparison
- business recommendations
- AI-assisted workflow summary

## Why This Exists
The project originally did not include Power BI assets. This dashboard provides a visual portfolio alternative that works directly from the repository.
